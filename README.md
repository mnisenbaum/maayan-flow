# MaaYAn Flow

**O futuro começa agora.**

Landing page e site do **MaaYAn Flow**, curso de tecnologia e inteligência artificial para adolescentes de 13 a 17 anos, realizado pelo [Maayan — Centro Cultural Judaico](https://maayan.rio), em Copacabana, Rio de Janeiro.

> Preparamos mentes brilhantes para um futuro onde a única constante é a inovação.

🔗 **Site no ar:** [maayan.rio/flow](https://maayan.rio/flow)

---

## Sobre o curso

O MaaYAn Flow ensina adolescentes a criar, aprender e desenvolver novas habilidades com tecnologia e IA — passando por fundamentos de programação (HTML, CSS, JavaScript) até o uso prático de ferramentas de inteligência artificial.

## Como rodar localmente

```bash
# clonar o repositório
git clone https://github.com/mnisenbaum/maayan-flow.git
cd maayan-flow

# instalar dependências
npm install

# rodar em modo desenvolvimento
npm run dev
```

O site abre por padrão em `http://localhost:3000`.

```bash
# build de produção
npm run build
npm run start
```
